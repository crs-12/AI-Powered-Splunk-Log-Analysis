# AI-Powered Log Analysis using Splunk

## Overview
This project integrates Splunk SIEM with Python and AI-driven analysis to automate log monitoring, threat detection, and incident response recommendations. The system processes security logs, identifies suspicious activities, and generates human-readable insights for analysts.

## Objectives
- Centralize security log collection using Splunk.
- Detect anomalies and suspicious events.
- Automate log analysis using Python.
- Generate AI-assisted incident summaries and recommendations.

## Tools & Technologies
- Splunk Enterprise
- Python
- Streamlit
- Splunk SDK
- Generative AI / LLM
- Windows Environment

## Methodology
1. Collect and index security logs in Splunk.
2. Search and retrieve logs using SPL queries.
3. Process logs using Python.
4. Perform tokenization and masking of sensitive information.
5. Generate AI-powered security insights.
6. Display results through an interactive dashboard.

## Key Features
- Automated log analysis
- Security event detection
- AI-generated incident summaries
- Dashboard visualization
- Sensitive data masking

## Project Architecture
Logs → Splunk → Python Processing → AI Analysis → Dashboard

## Results
Successfully automated security log analysis and reduced manual effort required for threat investigation and incident response.
